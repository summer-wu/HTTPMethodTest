#说明
在国外，对国内的服务器发HTTP DELETE请求，会被拦截，返回403 forbidden。

为了测试各种请求方法，做了这个app。

#页面结构：

+ 上面是三个按钮Get、Post、Delete。
+ 中间是要请求的url
+ 下面是返回的结果

#Delete请求收不到的解决方式

1、改成post或get请求

2、改成https连接
